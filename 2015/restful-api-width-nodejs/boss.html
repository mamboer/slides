MBTI性格分析

Myers-Briggs Type Indicator

是一种性格测试工具，用以衡量和描述人们在获取信息、作出决策、对待生活等方面的心理活动规律和性格类型

定位

  1. 你愿意出去找乐子(E)
  2. 或者你宁愿呆在家里读一本好书或看电视吗？(I)

悟性

  1. 当你学习新东西时，你是所见即所学(S)
  2. 或根据自己的直觉分析并产生印象？(N)

决定

  1. 当你对某件事作出决定时，你依靠逻辑推理(T)
  2. 还是倾向于情感和直觉？(F)

态度

  1. 你愿意自己生活的各个方面都井井有条，预先安排(J)
  2. 还是喜欢突如其来的惊喜？(P)
